## WifiAssit
  The easy way to use Wifi through PC.

## Install
  ```shell
  git clone https://github.com/lzjqsdd/WifiAssist.git
  cd WifiAssist`
  sudo ./install.sh
  ```
  Then edit config.ini,set PATH="/bin/wifi"
## Uninstall
  ```shell
  ./uninstall.sh
  ```
